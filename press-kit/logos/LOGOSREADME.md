# Logo files

## Which logo should I use?

| Scenario | Recommended file |
|----------|------------------|
| Website, light backgrounds | `svg/ret-logo-landscape-dark.svg` |
| Dark backgrounds, overlays | `svg/ret-logo-landscape-light.svg` or `png/square-white.png` |
| Print materials | SVG files (scalable) or high-res PNGs |
| Social media, thumbnails | `png/square-medium-400x300.png` or `png/square-small-200x150.png` |
| Wide/banner formats | `png/Red Eagle Tech Landscape Logo 2000x500.png` |
| Favicon | `favicon.ico` |

## Formats available

- **SVG** – Scalable vectors, ideal for web and print
- **PNG** – Raster images at various sizes, transparent backgrounds
- **ICO** – Favicon for websites
- **WebP** – Mascot images for web use

## Basic rules

- Maintain clear space around the logo
- Don't stretch, rotate, or alter the logo
- Don't change the colours
- Don't place the dark logo on dark backgrounds (use the white version)

For complete brand guidelines, see the [brand-guidelines folder](../brand-guidelines/).
